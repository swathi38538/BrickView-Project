# BrickView - Real Estate Intelligence Platform

## Project Overview
BrickView is a Streamlit-based real estate analytics application built using Python, SQLite, SQL, and Pandas.

## Features
- Property Filters & Explorer
- CRUD Operations
- SQL Query Explorer (30 SQL Queries)
- Real Estate Analytics Dashboard

## Technologies Used
- Python
- Streamlit
- SQLite
- Pandas
- Plotly

## Dataset
- Listings
- Properties
- Agents
- Buyers
- Sales

## Run the Project

```bash
streamlit run app.py
```
